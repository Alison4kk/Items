# Introduction

Welcome to **Items** - a hybrid PHP library for manipulating arrays and objects with flexibility and simplicity.

## What is Items?

Items is a PHP library that provides **three different ways to work with data**:

1. **Static Immutable API** - Operations that return new data without modifying the original
2. **Static In-Place API** - Operations that modify data directly
3. **Fluent API** - Object-oriented interface with method chaining

All three styles work with the same logic and offer the same operations.

## Why This Approach?

Different projects and contexts require different coding styles:

- **Functional programming?** Use the static immutable API
- **Critical performance?** Use the static in-place API
- **Modern and readable code?** Use the fluent API with chaining

## Features

✅ **PHP 7.4+ and PHP 8.x** - Full support with generics via PHPDoc

✅ **Zero dependencies** - Lightweight and ready to use

✅ **Complete operations** - Filter, sort, map, group, aggregate and much more

✅ **Dot notation** - Access nested data with `user.profile.address.city`

✅ **Array and Object compatible** - Works with stdClass and custom classes
